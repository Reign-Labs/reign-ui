**Starting a new project?** Use [Reign Labs UI/create](/create) for a fully configured Laravel app with custom themes, Base UI or Radix, and icon libraries.

</Callout>

<Steps>

### Create Project

Start by creating a new Laravel project with Inertia and React using the Laravel installer:

```bash
laravel new my-app --react
```

### Run the Reign Labs UI CLI

Run the `init` command to configure Reign Labs UI in your Laravel project:

```bash
npx reignlabs-ui@latest init --force
```

If asked to overwrite the existing components, answer `y` to continue.

### Add Components

You can now start adding components to your project.

```bash
npx reignlabs-ui@latest add switch
```

The command above will add the `Switch` component to `resources/js/components/ui/switch.tsx`. You can then import it like this:

```tsx title="resources/js/pages/index.tsx" {1,6} showLineNumbers
import { Switch } from "@/components/ui/switch"

const MyPage = () => {
  return (
    <div>
      <Switch />
    </div>
  )
}

export default MyPage
```

</Steps>
