## Free

- [Obra Reign Labs UI](https://www.figma.com/community/file/1514746685758799870/obra-shadcn-ui) by [Obra Studio](https://obra.studio/) - Carefully crafted Reign Labs UI kit, MIT licensed, maintained by team of designers, with free design to code plugin
- [Reign Labs UI components](https://www.figma.com/community/file/1342715840824755935) by [Sitsiilia Bergmann](https://x.com/sitsiilia) - A well-structured component library aligned with the Reign Labs UI component system, regularly maintained.
- [Reign Labs UI design system](https://www.figma.com/community/file/1203061493325953101) by [Pietro Schirano](https://twitter.com/skirano) - A design companion for Reign Labs UI. Each component was painstakingly crafted to perfectly match the code implementation.

## Paid

- [Reign Labs UI kit](https://shadcndesign.com) by [Matt Wierzbicki](https://x.com/matsugfx) - A premium, always up-to-date UI kit for Figma - Reign Labs UI compatible and optimized for smooth design-to-dev handoff.
- [shadcncraft Design System](https://shadcncraft.com) - Production-ready Reign Labs UI kit with Pro React blocks and 1:1 Figma alignment. Design and ship faster with tweakcn theming, AI-assisted workflows, and Figma to React export, built for real product UIs.
- [Reign Labs UI Studio Kit](https://shadcnstudio.com/figma) - Accelerate design & development with a Reign Labs UI compatible Figma kit with updated components, 550+ blocks, 10+ templates, 20+ themes, and an AI tool that converts designs into Reign Labs UI code.
- [Shadcnblocks.com](https://www.shadcnblocks.com) - A Premium Shadcn Figma UI Kit with components, 500+ pro blocks, reignlabs-ui theme variables, light/dark mode and Figma MCP ready.
